# Birincil Pazar Yatırımcısı

Sistemin **birincil talep tarafı**. KOBİ'nin listelediği token'ı doğrudan satın alıyor.

## Profil

- **Tipik örnek:** Berlin'de Hans, Dubai'de Sara, İstanbul'da Mehmet — bireysel kripto yatırımcısı
- **Bütçe:** 1.000$ – 100.000$
- **Web3 deneyimi:** Orta-yüksek — DeFi, RWA tokenları kullanmış
- **Risk profili:** Yüksek getiri (yıllık %15-25) için orta risk kabul

## Motivasyon

- DeFi APY'lerinden daha **stabil getiri**
- Gerçek dünya gelirine bağlı (RWA — Real World Asset)
- 60-120 gün gibi **kısa vade**
- USDC bazlı, döviz volatilitesi yok
- 100€'dan başlayan **fraksiyonel yatırım**

## Sistem İçindeki Görevleri

1. Wallet bağla
2. KYC tamamla (yatırımcı sınıfı: retail / accredited)
3. USDC yatır
4. Marketplace'i tara — alıcı rating, vade, beklenen yıllık getiri filtrele
5. Limit bid yerleştir veya market'ten al
6. Eşleşen emirden token'ı al
7. Vade dolana kadar bekle veya ikincil piyasada sat

## Karar Verirken Baktığı Sinyaller

- **Alıcı kredi notu** (A/B/C/D)
- **Vade** (kalan gün)
- **İskonto** (nominal vs ask price)
- **Yıllık efektif getiri** (annualize edildiğinde APY)
- **Likidite** (ikincil piyasa derinliği)
- **Ülke riski** (alıcının bulunduğu ülke)

## Ekranlar

- `/marketplace` — Tüm aktif faturalar, filtre + sort
- `/invoice/[id]` — Detay, order book, "Buy now" + "Limit bid"
- `/portfolio` — Tutulan tokenlar, beklenen getiri, vade takvimi
- `/wallet` — USDC bakiye, deposit/withdraw

## Smart Contract Etkileşimi

| İşlem | Contract | Fonksiyon |
|---|---|---|
| USDC approve | `MockUSDC.sol` | `approve(escrow, amount)` |
| Bid yerleştir | `OrderBook.sol` | `placeBid(invoiceId, price, qty)` |
| Market buy | `OrderBook.sol` | `marketBuy(invoiceId, qty, maxPrice)` |
| Bid iptal | `OrderBook.sol` | `cancelOrder(orderId)` |

## KPI'lar

- **Average APY** — yatırımcı portföyü efektif yıllık getiri
- **Default exposure** — default olan token oranı
- **Active hold time** — vade öncesi satış oranı
- **Reinvestment rate** — payout sonrası yeni yatırım oranı
