# İkincil Pazar Trader'ı

Birincil pazardan veya başka bir trader'dan token alıyor; vade dolmadan kar için satıyor. Sistemin **likidite akışını besleyen** persona.

## Profil

- **Tipik örnek:** DeFi'de aktif trader, RWA arbitrage yapan kripto fonu
- **Bütçe:** 5.000$ – 1M$
- **Web3 deneyimi:** Yüksek
- **Risk profili:** Hızlı giriş-çıkış, fiyat hareketinden kar

## Motivasyon

- **Time discount arbitrajı:** İlk yatırımcı 95€'dan aldı, 60. günde 97.5€'ya satıyor — 60 günde %2.6 getiri
- **Vade yaklaştıkça** token nominal'e yakınsıyor (zamanın değeri)
- **Yanlış fiyatlamadan** kar (örn. acil nakit isteyen yatırımcı düşük satıyor)
- **Olay tabanlı arbitraj:** Alıcı rating'i değişirse tepki

## Sistem İçindeki Görevleri

1. Birincil pazardan veya ikincil pazardan token al
2. Portföyünde tutarken fiyat hareketini izle
3. İkincil piyasada ask emir aç
4. Eşleştiğinde USDC al, kar realize et

## Stratejiler

| Strateji | Mantık | Risk |
|---|---|---|
| **Buy & hold to maturity** | %5 iskontoyla al, 90 gün bekle, %5.5 getiri | Default riski |
| **Time-decay arbitraj** | 30 günde %2 kar, çık | Likidite riski |
| **Rating arbitrajı** | Alıcı upgrade olunca al | Bilgi avantajı şart |
| **Kriz alımları** | Panic-sell yapan yatırımcılardan ucuza topla | Sistemik risk |

## Ekranlar

- `/marketplace` — Sort by "discount", "APY", "time-to-maturity"
- `/invoice/[id]` — Detaylı order book + trade history grafiği
- `/portfolio` — Pozisyon listesi, unrealized P&L
- `/orders` — Aktif limit emirleri

## Smart Contract Etkileşimi

İkincil piyasa **birincil ile aynı `OrderBook.sol` kontratını** kullanır. Tek fark `seller` parametresinin KOBİ değil, daha önceki yatırımcı olması.

```solidity
// Ahmet Bey ilk satışta
placeAsk(invoiceId=42, price=95, qty=500, seller=Ahmet)

// Hans 60. günde ikincil ask
placeAsk(invoiceId=42, price=97.5, qty=120, seller=Hans)
```

## KPI'lar

- **Volume traded** — toplam alım-satım hacmi
- **Avg holding period** — ortalama tutma süresi
- **Realized P&L** — kapatılan pozisyonlardan net kar
- **Hit rate** — karlı kapanan trade oranı
