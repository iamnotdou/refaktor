# Default Akışı

Lidl 30 gün geçtikten sonra hala ödemedi. Sistem default ilan ediyor, sigorta havuzu tazminat ödüyor.

## Aktörler

- **Platform Admin** (default'u tetikler)
- **InsurancePool.sol**
- **Escrow.sol**
- **Token sahipleri** (zarar görenler)

## Akış

```
1. Vade + 30 gün geçti, ödeme yok
2. Admin: escrow.declareDefault(invoiceId=42)
3. Default state aktivasyonu:
   - İkincil piyasada bu invoice tüm emirleri iptal
   - Token transferi yasak (frozen)
4. Sigorta hesabı:
   total = 50.000 USDC (nominal)
   recovery = overcollateralization tampon = 50.000 × 0.85 = 42.500 ödendi başında
   → 7.500 USDC sigorta tarafından karşılanmalı (ya da hepsini)
5. InsurancePool.payClaim(invoiceId=42, amount)
6. Token sahiplerine pro-rata dağıtım
7. Alıcı (Lidl) gelecek default rating'e D yapılır
```

## Sigorta Geri Kazanım Modeli

Faturanın **%80-85'i tokenize edilir**, **%15-20** tampon olarak tutulur:
```
50.000€ fatura
  ├── 42.500€ tokenize (500 × 85)
  └── 7.500€ tampon (overcollateralization)
```

Default durumunda:
- KOBİ'den toplam 42.500 USDC alındı (Ahmet'e gitti)
- Yatırımcılar 42.500 USDC kazanım bekliyordu
- Lidl ödemedi → Yatırımcılara sigorta havuzundan 42.500 USDC ödenir

## Senaryo Tablosu

| Lidl ödemesi | Sigorta havuzu | Yatırımcı net |
|---|---|---|
| 50.000€ tam | 0 | %100 nominal (eksiksiz) |
| 45.000€ kısmi | 5.000€ | %100 nominal (sigorta dolduruyor) |
| 0 (tam default) | 42.500€ | %100 (sigorta tüm yatırımcı payını karşılıyor) |
| Sigorta havuzu yetersiz | Pro-rata | Kayıp paylaşımı |

## Sigorta Yetersizliği (Worst Case)

Eğer sigorta havuzu defaulu karşılamak için yetersizse:
1. Mevcut havuz pro-rata dağıtılır
2. Eksik kalan kısım için "loss certificate" mint edilir
3. Sonraki dönemlerde havuz büyüdükçe loss cert sahiplerine ödeme yapılır
4. KOBİ Ahmet kara listeye alınır, yasal süreç başlar

## Yasal Boyut

MVP'de smart contract default'ı ilan ediyor ama gerçek hukuki recourse yok. Phase 2'de:
- Türkiye'de tahsilat avukatı entegrasyonu
- Uluslararası alıcılar için ICC arbitration
- Kleros benzeri merkeziyetsiz tahkim opsiyonu

## Game Theory Etkisi

Default oluştuğunda:
- Sigorta havuzu küçülür → APY azalır → stake'çiler unstake eder
- Alıcı rating düşer → o alıcının yeni faturaları daha yüksek iskontoyla satılır
- KOBİ rating düşer → o KOBİ yeni fatura listeleyemez
