## Smart Contracts (Polygon Amoy)

`contracts/scripts/deploy.ts`:
```typescript
import { ethers } from "hardhat";

async function main() {
  // 1. Mock USDC
  const USDC = await ethers.deployContract("MockUSDC");
  await USDC.waitForDeployment();
  console.log("USDC:", await USDC.getAddress());

  // 2. InvoiceNFT
  const NFT = await ethers.deployContract("InvoiceNFT");
  await NFT.waitForDeployment();

  // 3. InsurancePool
  const Pool = await ethers.deployContract("InsurancePool", [USDC.target]);
  await Pool.waitForDeployment();

  // 4. Escrow
  const Escrow = await ethers.deployContract("Escrow", [USDC.target, NFT.target, Pool.target]);
  await Escrow.waitForDeployment();

  // 5. OrderBook
  const Book = await ethers.deployContract("OrderBook", [NFT.target, Escrow.target]);
  await Book.waitForDeployment();

  // 6. Wire up roles
  await NFT.grantRole(await NFT.MINTER_ROLE(), Book.target);
  await Escrow.grantRole(await Escrow.RELAYER_ROLE(), Book.target);
  await Pool.grantRole(await Pool.ESCROW_ROLE(), Escrow.target);

  console.log("Deployed:");
  console.log("  USDC:", USDC.target);
  console.log("  InvoiceNFT:", NFT.target);
  console.log("  Escrow:", Escrow.target);
  console.log("  OrderBook:", Book.target);
  console.log("  InsurancePool:", Pool.target);
}

main();
```

Çalıştırma:
```bash
cd contracts
pnpm hardhat run scripts/deploy.ts --network amoy
```

`hardhat.config.ts`:
```typescript
networks: {
  amoy: {
    url: process.env.POLYGON_AMOY_RPC,
    accounts: [process.env.DEPLOYER_PRIVATE_KEY],
    chainId: 80002,
  },
}
```

## Frontend (Vercel)

```bash
cd frontend
pnpm build
```

`vercel.json`:
```json
{
  "framework": "vite",
  "buildCommand": "pnpm build",
  "outputDirectory": "dist",
  "env": {
    "VITE_BACKEND_URL": "@backend-url",
    "VITE_CHAIN_ID": "80002"
  }
}
```

Deploy:
```bash
npx vercel --prod
```

## Backend (Railway)

```bash
cd backend
railway login
railway init
railway link
railway up
```

`railway.toml`:
```toml
[build]
builder = "nixpacks"

[deploy]
startCommand = "node dist/index.js"
healthcheckPath = "/health"
restartPolicyType = "always"
```


## Verifikasyon

Polygonscan'de contract verify:
```bash
pnpm hardhat verify --network amoy 0xCONTRACT_ADDR "constructor_arg1" "constructor_arg2"
```

Bu jüri ekranında "verified" badge gösterir → güven artar.

## Rollback Stratejisi

Bug bulduysan:
1. Yeni deploy yap (yeni addresses)
2. Frontend env vars güncelle
3. Backend env vars güncelle
4. Eski deploy'u Vercel'de revert et
