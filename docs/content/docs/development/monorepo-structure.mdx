## pnpm Workspace

`/refaktor/pnpm-workspace.yaml`:
```yaml
packages:
  - 'docs'
  - 'frontend'
  - 'backend'
  - 'shared'
```

Not: `contracts/` submodule olduğu için workspace'e dahil **değil** — kendi `package.json`'ı ile bağımsız.

## ABI Sync Script

`scripts/sync-abi.js`:
```javascript
const fs = require('fs');
const path = require('path');

const contracts = ['InvoiceNFT', 'OrderBook', 'Escrow', 'InsurancePool', 'MockUSDC'];
const src = './contracts/artifacts/contracts';
const dst = './shared/abi';

for (const c of contracts) {
  const json = require(path.join(src, `${c}.sol/${c}.json`));
  fs.writeFileSync(
    path.join(dst, `${c}.json`),
    JSON.stringify({ abi: json.abi }, null, 2)
  );
  console.log(`✓ ${c}`);
}
```

Kullanım:
```bash
# Contract değişti?
cd contracts && pnpm hardhat compile
cd ..
pnpm sync:abi
git add shared/abi && git commit -m "abi: sync"
```

## TypeScript Config

`/refaktor/tsconfig.base.json`:
```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "Bundler",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "paths": {
      "@refaktor/shared/*": ["./shared/*"]
    }
  }
}
```

Her workspace `extends` ile bunu kullanır:
```json
// frontend/tsconfig.json
{
  "extends": "../tsconfig.base.json",
  "include": ["src"]
}
```

## Build Pipeline

```bash
# Tüm pakageleri build et
pnpm build

# Sırasıyla:
# 1. shared (en aşağıda, herkesin bağımlılığı)
# 2. backend (shared'a bağlı)
# 3. frontend (shared'a bağlı)
# 4. docs (bağımsız)
```

## CI Pipeline (GitHub Actions)

`/refaktor/.github/workflows/ci.yml` (Phase 2'de):
```yaml
name: CI
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          submodules: recursive
      - uses: pnpm/action-setup@v3
      - run: pnpm install
      - run: pnpm build
      - run: pnpm test
      - run: pnpm lint
```
