# Lokal Kurulum

## Önkoşullar

```bash
node -v   # >= 20
pnpm -v   # >= 9
git -v
```

Yoksa:
- Node: https://nodejs.org
- pnpm: `npm i -g pnpm`

## Klon ve Submodule

```bash
git clone https://github.com/iamnotdou/refaktor.git
cd refaktor

# Contracts submodule'unu çek (arkadaş repo URL'sini verecek)
git submodule update --init --recursive
```

## Dependency

```bash
# Root'tan tüm workspace'lerin paketlerini install et
pnpm install
```

## Environment

## Servisleri Çalıştır

### Tek Tek

```bash
# Terminal 1 — docs
cd docs && pnpm dev          # http://localhost:3000

# Terminal 2 — frontend
cd frontend && pnpm dev      # http://localhost:5173

# Terminal 3 — backend
cd backend && pnpm dev       # http://localhost:4000

# Terminal 4 — local hardhat node (opsiyonel)
cd contracts && pnpm hardhat node
```

### Hep Birlikte

```bash
pnpm dev:all
```

1. Network ekle:
   - Network name: Polygon Amoy
   - RPC: https://rpc-amoy.polygon.technology
   - Chain ID: 80002
   - Symbol: MATIC

2. Faucet'ten testnet MATIC al:
   - https://faucet.polygon.technology

3. Mock USDC al:
   - Frontend'de `/wallet` → "Get test USDC" → contract `faucet()` çağırır

## Troubleshooting

### "Cannot connect to MongoDB"
- MongoDB çalışıyor mu? `brew services list`
- URI doğru mu? `mongodb://localhost:27017`

### "Wallet wrong network"
- Frontend network switch modal otomatik gelir
- Manuel: MetaMask → Networks → Polygon Amoy

### "Insufficient gas"
- Faucet'ten test MATIC al
- Polygon Amoy çok ucuz, ~0.01 MATIC yeterli

### "Tx reverted"
- USDC approve verildi mi?
- Yeterli token bakiyesi var mı?
- Backend logs'a bak — relayer error?

### "WebSocket connection failed"
- Backend running mı? `curl localhost:4000/health`
- CORS ayarı `localhost:5173` izinli mi?
