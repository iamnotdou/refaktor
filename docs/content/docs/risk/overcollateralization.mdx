# Overcollateralization

Faturanın **%80-85'i** tokenize edilir; **%15-20'si** tampon olarak tutulur.

## Mekanik

```
Fatura nominal: 50.000€
Tokenize: 50.000 × 0.85 = 42.500€ (token sahiplerine vadeli ödenecek)
Tampon: 50.000 × 0.15 = 7.500€

Token sayısı: 500
Token nominal değeri: 42.500 / 500 = 85€

Yatırımcılar bu token'ları 75-85€ aralığında alır (iskontolu)
Vade: Lidl 50.000€ öder
  → 42.500 token sahiplerine
  → 7.500 → KOBİ'ye gider (eğer default yoksa)
```

## Default Senaryosu

Lidl 0 öder (worst case):
- Tampon yok → token sahiplerinin ödemesi sigortadan
- KOBİ'nin %15 tamponu → Sigorta havuzuna **subrogation alacağı** olarak ekleniyor

Aslında tampon **token sahipleri lehine** çalışıyor — onlar zaten 42.5k bekliyor, 50k'lık fatura ödendiğinde 7.5k fazla var → bu fazlanın %85'i token sahiplerine, %15'i KOBİ'ye gibi düşünülebilir.

## Why %15-20?

Ampirik:
- Türkiye ihracat default oranı: ~%2-5
- A rating buyer default: ~%0.5
- Ortalama recovery (yasal süreç): ~%30-40

Yani worst case kayıp:

```
kayıp = default_rate × (1 - recovery) = 5% × 65% = 3.25%
```

%15 tampon bu kaybın **4.6 katı** — yeterli güvenlik marjı.

## Game Theoretic Etkisi

KOBİ %15 kendi parasını masada bırakıyor → **skin in the game**.
- Sahte fatura çıkarsa %15 kaybeder
- Buyer'ı dikkatli seçer
- KOBİ ile yatırımcı çıkarları **aynı yönde** — buyer ödemeli

## Tampon Bittiğinde

Tampon **rüzgar perdesi**, sigorta **ana sigorta**:
1. Default → önce KOBİ'nin tamponu yenir (KOBİ %15 kaybeder)
2. Hala eksik varsa sigorta havuzu devreye girer

Bu **iki katmanlı** koruma yatırımcıyı daha güvenli yapar.

## Phase 2 — Dynamic Overcollateralization

Buyer rating'e göre değişken oran:
- A rating: %10 tampon yeterli
- B rating: %15
- C rating: %25 (mevcut sistem kabul etmiyor ama Phase 2'de daha riskli kabul edilirse)

```typescript
function getCollateralRatio(rating: string) {
  return { A: 0.10, B: 0.15, C: 0.25 }[rating] ?? 1.0;
}
```

## Rakip Karşılaştırma

- **Polytrade:** %5-10 overcollateral
- **Centrifuge:** Pool bazlı, sabit oran yok
- **Goldfinch:** %20-30 (yüksek risk LP'lerden)
- **FaturaChain MVP:** %15-20
