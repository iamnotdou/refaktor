# Escrow Mekanizması

Geleneksel faktoringde **factor** parayı tutar — güven gerekir. FaturaChain'de **smart contract** tutar — kod karar verir.

## Lock Akışı

```
1. Yatırımcı bid yerleştirir → USDC otomatik Escrow'a kilitli (yetkili approve)
2. Bid eşleşmedi → Escrow'da kalır (cancel ile geri alınır)
3. Bid eşleşti → Escrow seller'a transfer eder
4. Vade dolunca buyer Escrow'a öder → token sahiplerine dağıtılır
```

## Escrow vs Custody

| Geleneksel custody | FaturaChain Escrow |
|---|---|
| Banka veya factor parayı tutar | Smart contract tutar |
| Banka iflas ederse risk | Kontrat hatası dışında risk yok |
| Yetki bankada | Yetki kontratta (immutable) |
| Gizlilik var | Public on-chain |

## Bid Lock Mekaniği

```solidity
function placeBid(uint256 invoiceId, uint256 price, uint256 qty) external {
    uint256 lockAmount = price * qty;
    usdc.transferFrom(msg.sender, address(this), lockAmount);
    locked[msg.sender][invoiceId] += lockAmount;
    
    // Order book'a kaydet
    orderBook.placeBid(invoiceId, price, qty, msg.sender);
}
```

Yatırımcı bid'i iptal ederse:
```solidity
function cancelBid(uint256 orderId) external {
    Order memory order = orderBook.getOrder(orderId);
    require(order.maker == msg.sender);
    
    uint256 refund = order.remainingQty * order.price;
    usdc.transfer(msg.sender, refund);
    locked[msg.sender][order.invoiceId] -= refund;
    
    orderBook.cancelOrder(orderId);
}
```

## Trade Settlement

Match olduğunda:
```solidity
function executeTrade(...) external onlyRelayer {
    uint256 totalUsdc = qty * price;
    uint256 platformFee = totalUsdc * 100 / 10000;  // 1%
    uint256 insuranceFee = totalUsdc * 50 / 10000;  // 0.5%
    uint256 sellerNet = totalUsdc - platformFee - insuranceFee;
    
    // Buyer'dan locked → seller + fees
    locked[buyer][invoiceId] -= totalUsdc;
    usdc.transfer(seller, sellerNet);
    usdc.transfer(treasury, platformFee);
    insurancePool.fundFromFee(insuranceFee);
    
    // Token transfer
    invoiceNFT.safeTransferFrom(seller, buyer, tokenId, qty, "");
}
```

## Vade Ödemesi

```solidity
function markPaid(uint256 invoiceId, uint256 amount) external onlyOwner {
    Invoice memory inv = invoiceNFT.getInvoice(invoiceId);
    require(block.timestamp >= inv.dueDate, "Not yet due");
    
    usdc.transferFrom(msg.sender, address(this), amount);
    
    // Distribute pro-rata
    address[] memory holders = getTokenHolders(invoiceId);
    for (uint i = 0; i < holders.length; i++) {
        uint256 balance = invoiceNFT.balanceOf(holders[i], invoiceId);
        uint256 share = amount * balance / invoiceNFT.totalSupply(invoiceId);
        usdc.transfer(holders[i], share);
    }
    
    // Burn tokens
    invoiceNFT.burnAll(invoiceId);
}
```

## Default

```solidity
function declareDefault(uint256 invoiceId) external onlyOwner {
    Invoice memory inv = invoiceNFT.getInvoice(invoiceId);
    require(block.timestamp >= inv.dueDate + 30 days, "Wait 30d grace");
    
    // Sigorta'dan tazminat çek
    uint256 nominalSold = invoiceNFT.totalSupply(invoiceId) * 100; // 100€/token
    insurancePool.payClaim(invoiceId, nominalSold);
    
    // Token sahiplerine dağıt
    distributeFromInsurance(invoiceId);
    
    // Subrogation: buyer alacak hakkı pool'a
    pool.recordSubrogation(invoiceId, buyer, nominalSold);
}
```

## Pause Mechanism

Acil durumda admin pause edebilir:
```solidity
function pause() external onlyAdmin {
    _pause();
}
```

Pause iken:
- Yeni bid/ask kabul edilmiyor
- Trade settlement kabul edilmiyor
- Mevcut bid'ler iptal edilebilir (kullanıcı parasını alabilir)
- Vade ödemesi normal devam ediyor
