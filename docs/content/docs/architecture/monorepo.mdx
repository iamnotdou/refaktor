# Monorepo Yapısı

```
refaktor/
├── docs/                  Fumadocs — bu dokümantasyon
├── contracts/             Solidity (git submodule, ayrı repo)
├── frontend/              React + Vite + wagmi + tailwind
├── backend/               Node + Express + Mongoose
├── shared/                ABI, types, constants — npm workspace
└── README.md
```

## Workspace Setup (pnpm)

`/refaktor/package.json` (root):
```json
{
  "name": "refaktor",
  "private": true,
  "workspaces": ["docs", "frontend", "backend", "shared"],
  "scripts": {
    "dev:docs": "pnpm --filter docs dev",
    "dev:frontend": "pnpm --filter frontend dev",
    "dev:backend": "pnpm --filter backend dev",
    "dev:all": "pnpm -r --parallel dev"
  }
}
```

## Submodule (contracts)

`contracts/` ayrı bir repo. Arkadaş orada Hardhat ile çalışır, bu repo bir snapshot'a pin'lenir:
```bash
# İlk kurulum
git submodule add <CONTRACTS_REPO_URL> contracts
git submodule update --init --recursive

# Yeni contract versiyonuna güncelle
cd contracts && git checkout main && git pull
cd .. && git add contracts && git commit -m "bump contracts to <hash>"
```

## ABI Senkronizasyonu

Contracts ekibi her release'de:
```bash
# Contracts repo'da
pnpm hardhat compile
cp artifacts/contracts/**/*.json ../shared/abi/
git add ../shared/abi && git commit -m "abi: sync"
```

Frontend/backend `shared/abi/InvoiceNFT.json`'u import eder.

## Dev Yardımcı Script'ler

```bash
# Hep birlikte ayağa kalk
pnpm dev:all

# Sadece backend + frontend (contract submodule'da local node ayrı)
pnpm dev:frontend & pnpm dev:backend

# ABI'leri yeniden senkronize et
pnpm sync:abi
```
