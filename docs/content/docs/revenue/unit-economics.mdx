# Birim Ekonomisi

## Tek Fatura Senaryosu

Ahmet Bey'in 50.000€ Lidl faturası, 500 token, %1 platform fee + %0.5 sigorta + %0.25 ikincil komisyon.

## Birincil Satış Geliri

```
Tüm 500 token satıldı: 500 × 95€ = 47.500€
Platform fee: 47.500 × 1.0% = 475€
Sigorta payı: 47.500 × 0.5% = 237.5€
KOBİ alır: 47.500 - 712.5 = 46.787.5€
```

## İkincil Pazar Geliri

500 token toplam 3 kez el değiştiriyor (ortalama):
```
3 × 47.500 × 0.25% = 356.25€ ek platform geliri
```

## Toplam Platform Geliri (Tek Fatura)

```
Birincil platform fee: 475€
İkincil platform fee: 356€
Sigorta katkısı (ayrı havuz): 238€
Net platform: 475 + 356 = 831€
```

KOBİ'nin maliyet: %1.5 (geleneksel %15-20'ye karşı)

## Yatırımcı Tarafı

Hans 60. günde 120 token sattı (95 → 97.5):
```
Yatırım: 120 × 95 = 11.400€
Satış: 120 × 97.5 = 11.700€
Brüt kar: 300€
İkincil komisyon (Hans öder): 11.700 × 0.25% = 29.25€
Net Hans karı: 270.75€
60 günde %2.4 → yıllık %14.5 APY
```

Vadeye tutan yatırımcı (örn. Erkin):
```
Yatırım: 60 × 97.5 = 5.850€ (60. günde Hans'tan aldı)
Vade ödemesi: 60 × 100 = 6.000€
Net: 150€ kar
60 günde %2.6 → yıllık %15.5 APY
```

## Sigorta Stake'çisi Tarafı

```
Pool toplam 100.000€ stake
Bir fatura'dan sigorta payı: 238€
Aylık 100 fatura → 100 × 238 = 23.800€/ay sigorta geliri
Yıllık 285.600€

Stake'çi APY = 285.600 / 100.000 = %285 brüt (default'suz)

Default rate %0.5 olursa:
  Aylık 100 × 0.5% = 0.5 default × 50.000€ = 25.000€ default
  Sigorta tarafından ödenen yıllık: 300.000€
  Net stake'çi: 285.600 - 300.000 = -14.400€ (zarar)
```

`★ Insight ─────────────────────────────────────`
Stake'çinin karlı olması için **(default rate × nominal kayıp) < komisyon geliri** olmalı. Bu ancak buyer rating'i sıkı tutulursa mümkün. Bu yüzden A/B rating filtresi sadece yatırımcı için değil, **sigorta sürdürülebilirliği için kritik**.
`─────────────────────────────────────────────────`
