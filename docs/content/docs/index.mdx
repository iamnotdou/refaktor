# FaturaChain

**Tek cümlede:** Factoring'in Web3 versiyonu — açık pazar (order book) + ikincil piyasa ile fatura finansmanı.

Türkiye'de 1.87 trilyon TL'lik faktoring pazarı var. KOBİ'lerin %99.6'sı vadeli fatura bekliyor. İhracatta 90-120 gün ödeme bekleme süresi nakit akışını boğuyor. FaturaChain, bu faturayı NFT'ye dönüştürüp global yatırımcılara açık pazarda satılır hale getirir.
