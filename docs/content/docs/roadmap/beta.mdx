# Phase 2 — Beta

Hackathon → ilk 6 ay yol haritası.

## Hedefler

- 100 KOBİ pilot
- 1000 yatırımcı
- 5M EUR toplam volüm
- Mainnet hazır kontratlar (audit edilmiş)
- BDDK ön görüşmeleri

## Ana Özellikler

### Gerçek GİB e-Fatura API
- Mock'tan production GİB API entegrasyonu
- MFKS sorgusu (çift finansman önleme)
- Otomatik fatura doğrulama
- e-Müstahsil makbuzu desteği

### KYC/AML Tam Akış
- Sumsub veya Persona entegrasyonu
- Pasaport + selfie
- Adres doğrulama
- PEP/sanctions screening
- Türkiye için TC Kimlik no doğrulama

### AI Kredi Skorlama
- Buyer geçmiş davranışı (öğrenen model)
- Sektör + ülke + büyüklük features
- Real-time rating güncellemesi
- Otomatik ihale eşik kontrolü

### Tam İkincil Piyasa
- Limit + market + IOC + FOK + post-only emir tipleri
- Order history grafikleri (TradingView lite)
- Volume chart, depth chart
- Heatmap (yoğun saat görselleri)

### Fiat On/Off Ramp
- TL → USDC direkt (Türkiye için kritik)
- EUR → USDC (KOBİ için)
- BiLira veya başka bir Türk on-ramp ortaklığı

### Mobil Uygulama
- React Native (yatırımcı odaklı)
- Push notification (eşleşme, vade, default)
- Biometric login

### Otomatik Yatırım (AutoBidder)
- Kullanıcı kriter belirler (rating, APY, sektör)
- Bot otomatik bid yerleştirir
- Portföy rebalancing

### Chainlink Oracle
- Buyer ödeme verifikasyonu (banka API integration)
- Currency rate (EUR/USD/TL)
- Sektörel risk feed

### Fatura Bundling
- Birden fazla fatura paket halinde sat
- Risk dağılımı (1 default = paket %5 etki)
- Yatırımcı tek tıkla portföy kurar

## Teknik

### Mainnet Geçiş
- Polygon mainnet deploy (Amoy → Polygon PoS)
- Gerçek USDC (Circle)
- MultiChain support başlangıç (Arbitrum)

### Audit
- Trail of Bits veya OpenZeppelin
- Tahmini cost: 50-80k EUR
- 4-8 hafta süreç

### Backend Ölçeklendirme
- Multi-relayer matching (decentralization başlangıç)
- Redis cluster (matching engine)
- WebSocket sharding

### Monitoring
- Datadog veya Grafana stack
- On-chain event monitoring (Tenderly)
- Alert on suspicious volume
