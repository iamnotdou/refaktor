# MVP — Hackathon

**Etkinlik:** Bi-Thongo Web3 Hackathon  
**Mekan:** Biruni Teknopark  
**Süre:** 32 saat  
**Tarih:** Nisan 2026

## Ekip ve Roller

| Kişi | Sorumluluk |
|---|---|
| Doğu (sen) | Frontend + Backend + Demo + Pitch |
| Arkadaş | Smart Contracts + Test |

## 32 Saat Plan

### Saat 0-2: Kurulum (Ortak)
- Repo klon, dependency install
- Hardhat local node
- React dev server
- Backend Express server
- MetaMask local node bağla

### Saat 2-8: Foundation
**Sen:**
- Wagmi v2 + RainbowKit kurulumu
- Layout + routing
- Fatura yükleme formu (UI sadece, henüz contract yok)
- Backend skeleton + MongoDB connection
- Mongoose şemalar

**Arkadaş:**
- `InvoiceNFT.sol` (ERC-1155 mint)
- `Escrow.sol` skeleton
- Hardhat unit test (mint, transfer)

### Saat 8-14: Marketplace + Matching
**Sen:**
- `/marketplace` sayfası — kart layout
- `/invoice/[id]` — order book UI
- "Bid Ver" / "Listele" modal'ları
- Backend matching engine (basit FIFO)
- Mock GİB API endpoint

**Arkadaş:**
- `OrderBook.sol` (placeBid, placeAsk, executeTrade)
- `InsurancePool.sol` skeleton
- E2E test: invoice mint → list → trade

### Saat 14-20: Entegrasyon
**Sen:**
- Frontend ↔ contract entegrasyonu (ABI'ler hazır)
- WebSocket order book updates
- Wallet onboarding flow
- İkincil pazar UI
- Portfolio sayfası

**Arkadaş:**
- Contract'ları Polygon Amoy'a deploy
- `MockUSDC.sol` (faucet fonksiyonu ile)
- Default + claim akışı kontrat tarafı
- ABI'leri shared/abi/'ye export

### Saat 20-26: Test ve Bug Fix
**Birlikte:**
- E2E test: 4 demo cüzdan ile baştan sona akış
- Bug fix
- UI polish (loading states, errors, toasts)
- Mobile responsive check
- Network switch logic

### Saat 26-30: Demo Hazırlık
- Seed script — 3 fatura, 5 yatırımcı, 8 emir önceden ekle
- Demo cüzdanlarına test USDC dağıt
- Demo'yu 3 kez baştan oyna, kronometre
- Edge case'leri test (yetersiz bakiye, rejected tx, vs.)

### Saat 30-32: Pitch + Sunum
- Pitch deck (10 slayt)
- Sunum provası
- Q&A için olası sorular

## Risk Yönetimi (Hackathon İçi)

| Risk | Önlem |
|---|---|
| Contract bug → demo bozulur | Önceden 3× E2E test, fallback local node hazır |
| Polygon Amoy faucet kuruyor | 5 farklı faucet'tan testnet MATIC topla |
| Frontend ↔ contract uyumsuzluk | Sabah erken interface freeze |
| Demo gününde internet | Local node + offline demo hazır |
| Pitch unutmak | 3× rehearsal, kart kullan |
